mclib.hooker[setkey].mclib.hooker Class

Defined in: D:\Projects\Programming\OpenSource\mclib\mclib.js:914
Module: mclib

Class or Namespace that handles hooks. Hooks are like plugins. They are an architecture that allows predescribed external pieces of
code to be executed from your code. In other words, you need to document the hooks in order for them to be used. This is extremely useful if you using something like ExtJS and have a form where you want to allow a external developer using your application to create buttons and process button clicks from the form.

Hooks in their most basic form look like: - namespace - init() returns true or false - section - there can be more than one section - execute() that returns a value - there can be more than one execute function.
